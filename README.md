# Parse page

### Описание

Программа использует готовые данные из сервиса путешествий, придаёт нужный 
формат и выводит их пользователю. Можно ознакомиться с основными 
авиакомпаниями, совершающими перелёт из Санкт-Петербурга в Москву, а так же 
узнать от какой суммы начинаются цены на билет.

### Примечание

Следующие команды могут не подходить для вашей операционной системы. Данные 
инструкции выполнены на Windows OS.

### Как установить

- Клонировать репозиторий

```bash
git clone https://github.com/ShreddedCode/parse_page.git
```
- Перейдите в нужный каталог
```
cd parse_page
```
- Установить виртуальную среду
```
python -m venv venv
```
- Активировать виртуальную среду
```
source venv/Scripts/activate
```
- Выполнить установку всех зависимостей
```
pip install -r requirements.txt
```
- Запустить программу
```
python parse.py
```
### Технологии
Python 3.9
